# Anforderungen

Dieses Verzeichnis enthaelt alle Anforderungen fuer das Projekt googlemapscli.

## Nummernformat

Anforderungen verwenden das Format `RXXXXX` (5-stellig mit fuehrenden Nullen).

Dateinamen: `R[NNNNN]-[slug].md` (z.B. `R00001-user-authentication.md`)
